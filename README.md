# RuWeather-JavaConsole ☀️🌦️

RuWeather es una aplicación de consola en Java que permite consultar el clima de cualquier ciudad usando la **API de OpenWeather**. Los datos se procesan en **JSON** y se muestran de manera clara y organizada.

## Características

- Permite nombres de ciudades con espacios (ejemplo: "New York", "Los Angeles").
- Salida con emojis para mejorar la visualización del clima.
- Muestra:
  - Temperatura actual y sensación térmica
  - Humedad
  - Velocidad del viento
  - Descripción del clima
  - Horas de amanecer y atardecer
- Manejo de errores por ciudad no encontrada o problemas de conexión.
- Salida del programa escribiendo `F`.

## Cómo usar

1. Clonar o descargar el repositorio.
2. Compilar el proyecto en Java 11 o superior.
3. Ejecutar `WeatherCons.java`.
4. Escribir el nombre de la ciudad que quieres consultar.
5. Para salir, escribe `F`.
